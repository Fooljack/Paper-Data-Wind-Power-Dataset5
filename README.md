## Description of Public Dataset 5 in the Paper
The `Dataset5` folder contains 6 publicly available wind power-related datasets from different sites. Both DTSNet and baseline models utilize the data from the file `Wind farm site 1 (Nominal capacity-99MW).xlsx`.

For other detailed data descriptions, refer to the qualitative analysis in the paper: *"Solar and wind power data from the Chinese State Grid Renewable Energy Generation Forecasting Competition"* [1].

### Reference
[1] Chen, Y., Xu, J. Solar and wind power data from the Chinese State Grid Renewable Energy Generation Forecasting Competition. *Sci Data*, 9, 577 (2022). https://doi.org/10.1038/s41597-022-01696-6.
