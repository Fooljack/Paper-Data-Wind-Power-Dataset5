## Description of Public Dataset 5 in the Paper
Six publicly available wind power datasets for different sites are provided as separate files, with the data from *"Wind farm site 1 (Nominal capacity-99MW).xlsx"* used by both DTSNet and the baseline models.

For other detailed data descriptions, refer to the qualitative analysis in the paper: *"Solar and wind power data from the Chinese State Grid Renewable Energy Generation Forecasting Competition"* [1].
